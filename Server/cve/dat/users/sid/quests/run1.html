Learn how to run your programms from command shell.

openfile.
Steps
  1. Read about how to run files in java from the command shell.
  2. Demonstrate a successful run session.
Rewards
IDE:6, Java:1
Reading
First, start by opening an existing file (named Example2.java); click on the "File" menu, then choose "Open File" menu item. 
    class Example2 {
         public static void main(String[ args){
              int i = 3;
    	        i++;
    	        System.out.println(i);	// "4"
 	    	  ++i;			   
    		  System.out.println(i;	      // "5"
	    	  System.out.println(++i);	// "6"
    		  System.out.println(i++);	// "6"
    	  	  System.out.println(i);	// "7"
         }
    }
Then, compile your program by typing "javac Example2.java" in the shell message box. If the program is correct, you should see no error messages. Otherwise, go back and try to fix the the program and correct the errors. If you cleand your program from errors, then it will be ready for the running step. Then, run your program by typing "java Example2" in the shell message box. Then the output will appear in the message box.

Session
run: 
java Example2
