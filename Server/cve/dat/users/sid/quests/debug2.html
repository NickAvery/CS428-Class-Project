Steps
Follow the following steps to debug your program:
  1. ICI supports GNU gdb, the Sun JDK jdb, and the Uncon udb debuggers.
    The ICI interface to the debuggers has been designed to make the
    most common debugging tasks easy. The goal is to make using the
    native debuggers as easy as possible for all users.
    ICI accomplishes this by showing a command window (shell area)
    interface to the debugger. You can enter any debugger command in
    this window, and enjoy full access to all debugger features.
  2. To use ICI debugging features:
  3. Select "Start" from the "Debug" menu.
    4. - A "GDB", "JDB", or "UDB" prompt will appear in the commands(shell) area (depending on the type of the used program)
    - Your debugging session started and you have a full access to the all the debugger features. - start using the debugging commands.
