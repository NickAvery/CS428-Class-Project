Quiz (2/2)
How can you get a long listing that shows file permissions and size?
ls -l
How can you list all files in all subdirectories?
ls -R
Demo (2/2)
Please show me a simple listing of the root directory.
ls /
Can you show me a listing of the current directory, sorted by the time each file was last accessed?
ls -t
